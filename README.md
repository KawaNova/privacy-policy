# プライバシーポリシー

AozoraDev-tech（以下「当方」）は、当方が提供するスマートフォン向けアプリケーション（以下「本アプリ」）における、ユーザーの個人情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

## 1. 取得する情報および利用目的

本アプリでは、ユーザーの個人を特定できる情報（氏名、住所、電話番号、メールアドレス等）は取得しておりません。
ただし、本アプリでは広告配信およびアプリの利用状況解析のために、以下の第三者モジュールを利用しており、匿名化された情報（端末情報、IPアドレス、広告識別子等）が自動的に取得される場合があります。

取得された情報は、各社のプライバシーポリシーに基づいて管理されます。

**・Google AdMob（広告配信）**
提供者：Google LLC
目的：ユーザーに適した広告を配信するため
[Googleのプライバシーポリシー](https://policies.google.com/privacy)

**・Google Analytics for Firebase（利用状況解析）**（※利用する場合）
提供者：Google LLC
目的：アプリの利用状況を把握し、品質向上やバグ修正に役立てるため
[Googleのプライバシーポリシー](https://policies.google.com/privacy)

## 2. 個人情報の第三者提供について

当方は、取得した情報を、法令に基づく場合やユーザーの同意がある場合を除き、第三者に提供することはありません。

## 3. 免責事項

本アプリの利用により生じたトラブルや損害について、当方は一切の責任を負わないものとします。
本アプリからリンクやバナーなどによって他のサイトに移動された場合、移動先サイトで提供される情報、サービス等について一切の責任を負いません。

## 4. 著作権・肖像権等

本アプリ内に掲載されている写真やイラスト、文章等の著作権や肖像権は、各権利所有者に帰属します。

## 5. 本ポリシーの変更

当方は、法令の制定、改正等により、本ポリシーを適宜見直し、予告なく変更する場合があります。変更後のプライバシーポリシーは、本ページに掲載された時点で効力を生じるものとします。
